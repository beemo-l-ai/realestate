# src/lib

## 역할
- 공통 타입/설정/Oracle DB 접근/집계 로직

## 핵심 파일
- `config.ts`: 환경 변수, 수도권 기본 지역 코드
- `oracle.ts`: Oracle 풀/커넥션 관리
- `schema.ts`: 테이블/인덱스 초기화
- `types.ts`: 도메인 타입 정의
- `store.ts`: 거래 저장 및 월별 집계 생성/조회

## 이 디렉토리만 수정하면 되는 경우
- Oracle 테이블/인덱스 구조 변경
- 공통 타입 변경
- 집계 통계(평균/중앙값 등) 계산 규칙 변경

## 주의사항
- 스키마 변경 시 `README.md`와 동기화 필요
- 집계 함수는 가능한 순수 함수로 유지
